# 🔹 EXP 8: Task Assignment

## 📌 Definition
Task assignment is the process of allocating tasks to processors in a way that minimizes total execution cost.

---

## ⚙️ Approach Used
- Greedy method
- For each task, select processor with minimum cost

---

## ❓ Why this approach?
- Simple and efficient
- Works well for basic optimization problems
- Reduces execution time

---

## ✅ Advantages
- Easy to implement
- Fast execution
- Works well for small systems

---

## ❌ Disadvantages
- Not always globally optimal
- Doesn’t consider complex constraints
- Static (no dynamic adaptation)

---

## ▶️ How to Run

```bash
python task_assignment.py
